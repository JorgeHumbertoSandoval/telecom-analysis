# Telecom Analysis

## Objetivo
Analizar datos de clientes de ConnectaTel para identificar patrones de uso, segmentar usuarios y generar insights accionables para el negocio.

## Herramientas utilizadas
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Análisis realizado
- Limpieza y validación de datos
- Tratamiento de sentinels y fechas inválidas
- Detección de outliers
- Segmentación de clientes
- Visualización de distribuciones
- Insights ejecutivos

## Principales hallazgos
- La mayoría de los usuarios pertenece al plan Básico.
- El segmento predominante corresponde a adultos con uso medio.
- Los usuarios Premium muestran mayor presencia en consumos altos.
- Se identificó oportunidad para un plan intermedio entre Básico y Premium.
